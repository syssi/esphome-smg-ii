通讯数据格式

  • 通讯时数据以字(WORD— 2 字节)的形式回送，回送的每个字中，高字节在前，低字节在后，如果2 个字连续回送(如：长整形)，则高字在前，低字在后。

帧格式

  • 寄存器内容查询（功能码03H）

    查询的起始地址和结束地址必须是一个完整数据块的起始地址和结束地址，否则返回的数据不正确。例如：设备序列号的寄存器起始地址是186，长度是12，则查询时起始地址不能在186~198之间，同样结束地址（起始地址+读取的寄存器个数）不能落在这个范围。

警告码说明

系统警告是一个32位的无符号长整型，每一位对应一个警告，每一位也可以通过警告掩码I进行屏蔽，屏蔽后对应警告将不会在LCD读到，也不会通过命令读到。

电量查询

逆变器可以存储一定时间内的电量数据，包括PV的发电量和输出带负载的用电量，存储的电量，可以分为总电量、年电量、月电量、日电量、时电量。每种电量的存储时间范围见下

  • 发电量数据结构

  • 小时发电量

  • 日发电量(不存在的日期数据无效，例如4月31日)
  • 总发电量、年发电量、月发电量

以上发电量均可使用文件读取功能码(14H)读取

unsigned short sModbusCrc16(INT8U *chMsg, INT16U dataLen)
{ 
    unsigned char ubCRCHi = 0xFF;
    unsigned char ubCRCLo = 0xFF;
    unsigned char duwIndex;
    while (dataLen --)
    { 
        duwIndex = 0xff&(ubCRCHi ^ *chMsg++);
        ubCRCHi = 0xff&(ubCRCLo ^ auchCRCHi[duwIndex]); 
        ubCRCLo = auchCRCLo[duwIndex]; 
    } 
    return (ubCRCHi << 8 | ubCRCLo);
}
